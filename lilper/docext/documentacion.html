JUEGO DEL AHORCADO

Manual de usuario

1 Ingresa al juego a través de la página principal de Old Games, dando clic en el icono del Juego del ahorcado.


2 Se mostrará el juego.


3 En la parte superior derecha podrás ver las pistas para adivinar la palabra.


4 Para empezar a jugar debes dar clic con la flecha del mouse a cada letra.


5 Cada vez que aciertes una letra la podrás observar en las cajitas blancas que están arriba del teclado.


6 Por cada error que cometas irá apareciendo como se va ahorcando el muñequito solo puedes cometer 5 errores.


7 Cuando hayas completado la palabra satisfactoriamente se mostrará un mensaje de que has ganado o en caso contrario de que has perdido y te dirá cuál era la respuesta correcta.


8 Si quieres volver a jugar solo debes dar clic en el botón de Volver a Jugar.


9 Si quieres volver al menú principal dale clic en el botón HOME.



En la parte inferior podrás ver información del perfil del Autor del juego

Descripción del problema

Se elabora un software para un juego de ahorcado, donde los usuarios pueden adivinar las palabras utilizando pistas, sí acierta se le mostrará un mensaje de que ha logrado completar satisfactoriamente la misión. Cada error le restará puntos y al término de cinco errores perderá el juego el cual puede volver a iniciar dando clic en un botón. Para el efecto se utiliza la web para contener el software, haciendo uso de Html, JavaScript para el funcionamiento y CSS para dar una vista atractiva al usuario.

Descripción paso a paso

1Se ingresa al juego desde la página principal, con el que se utilizó un href para el hipervínculo de los archivos Html.


2 Se mostrarán las pistas en la esquina superior izquierda del canvas (lienzo) del juego. Para las cuales se hace uso de un Switch para controlar las pistas de las palabras.


3Se hace uso de un Function= teclado para distribuir el abecedario en el Canvas.


4 Para seleccionar las letras se hace el uso de un Function=seleccionar que permitirá reconocer o detectar las palabras elegidas al azar.


5 Mostrará la letra en una cajita si acierta sino mostrara la imagen del ahorcado que ira sumando según los errores que cometa máximo 5 que utiliza un Function=GameOver y un Function= Horca.


6Se utiliza un button type=reset para reiniciar el juego y volver a jugar.


7Se utiliza un button type=button para regresar al Home .


8Y un footer para los datos que contiene información del Creador de la página .

Manual de usuario


- Leer las instrucciones previas al juego.

- Se te presenta un cuadro de 9 casillas, debes marcar usando el mouse (Click izquierdo) tu casilla elegida.

- Después de marcar tu casilla, esperas a que la computadora conteste.

- Así consecutivamente hasta lograr formar una linea, horizontal, vertical o paralela.

- Cuando los turnos acaben, presiona volver a jugar para repetir la partida.

Descripción del problema


- Es una página virtual, con el único objetivo de entretener al público más infantil o personas que se quieran distraer un poco. El juego elaborado es llamado:

  "Juego del Gato" más conocido como ¡Totito!.

- El proyecto fue elaborado con una inmesidad de funciones y etiquetas, cada una con propósitos diferentes. El juego está diseñado para jugar y descubrir si

  eres rival digno de la inteligencia artificial.

- Es un juego de estrategia y un reto de lograr el objetivo deseado. Por lo tanto, se requiere de concentración y dedicarle un poco de tiempo, las formas de ganar

  son contadas y debes descubir cuáles son. ¿Estás preparado para adivinarlas? Prueba tu inteligencia...

Descripción paso a paso


Paso 1) Se crea el archivo html, junto al html:5.

Paso 2) Se ingresa en el body las diferentes etiquetas que conlleva la estructura del mismo.

Paso 3) Se crean dentro del body el button type reset y el acceso directo a la página principal.

Paso 4) Se agrega la etiqueta Script, para crear variables y las diferentes funciones que hacen funcionar el programa.

Paso 5) Se crea la etiqueta Style, para agregarle forma y diseño a la estructura.

Paso 6) Se crea el Footer junto a sus ramas en el Body, después de terminar el juego.

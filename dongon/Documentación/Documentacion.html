Planteamineto del Problema

Diseñar un programa con HTML, CSS y si es necesario, JavaScript. El cual muestre una ruta de acceso para otros cuatro programas, es decir, estructurar la página principal para un sitio web de juegos, desarrollado colaborativamente con otras cuatro personas.


Análisis

1Establecer una página principal a la cual el Usuario acceda fácilmente.

2Colocar un menú en la interface de la página principal.

3Vincular las otras páginas con la página principal para establecer una ruta de conexión.

4Darle un diseño interactivo a la página principal.

5Estructural el sitio web para juegos.

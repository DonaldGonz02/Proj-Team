Manual de usuario


- Leer las instrucciones para mejor comprención.

- Para moverse use las teclas [←], [→] y disparar con control ya sea derecho o izquierdo.

- Para ganar tiene que destruir todas las naves enemigas.

- Si llega a ser impactado por una bala o se le acaba las municiones perdera.

- Si quiere volver a jugar presione el boton "volver a jugar".

Descripción del problema


- Es una pagina creada con el objetivo de entretener y pasar el rato jugando un juego llamado:

  "Space Invaders" traducido al español como "Invasores espaciales".

- El siguiente proyecto fue elaborado con muchas funciones y etiquetas cada uno de estos juegos han sido creados con el

  proposito de entretener y poner a prueba tus habilidades

- Es un juego que consiste en eliminar todas las naves enemigas disparandoles en tu nave hasta eliminarlas a todas, pero ten cuidado ya que las naves enemigas

  trataran de eliminarte disparandote asi que tendras que esquivar sus balas pero cuidado porque si te quedas sin munición perderas.

Descrpción paso a paso


Paso 1) Se crea la carpeta html 5 y se agrega la etiqueta canvas.

Paso 2) Se crea la carpeta de css y se agregan los diseños.

Paso 3) Se agrega la carpeta de javascrip y se llenan algunas funciones y variables.

Paso 4) Se agregan todas las demas funciones para que funcione bien el juego.

Paso 5) Se le mejoran unos detalles en el estilo y de las funciones.

Paso 6) Se le agrega el footer.
